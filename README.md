Designed and developed a classic Pong game in x86 Assembly language for DOS using BIOS and DOS interrupts in VGA graphics mode (320×200). Implemented single-player mode with an AI-controlled paddle and a two-player mode, along with real-time keyboard input, pixel-based graphics rendering, AABB collision detection, score tracking, game state management (main menu, gameplay, and game-over screens), and smooth frame timing. The game also features dynamic ball physics, paddle movement constraints, and continuous gameplay with score updates.
