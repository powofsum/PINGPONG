Developed a classic Pong game in x86 Assembly language for DOS using BIOS and DOS interrupts in VGA graphics mode (320×200, 256 colors). The game features both single-player (AI-controlled opponent) and two-player modes, real-time paddle movement, ball physics with collision detection, score tracking, game-over and restart functionality, and an interactive main menu. Implemented low-level graphics rendering by directly plotting pixels, keyboard input handling through interrupts, frame timing control, and game state management without the use of external libraries or game engines.
